Up to date learning material based on open data

Problem:
Tryckta läromedel (böcker) blir snabbt inaktuella och är dyra för lärare att införskaffa.
Ibland används helt enkelt gamla böcker för att spara pengar, och då ökar risken för att en utvecklare en förlegat och felaktig uppfattning om världen och ens närliggande omgivning.

Tänk om det vore enkelt för lärare att kunna skriva ut dagsfärska faktablad med intressant innehåll som flera olika lärare kan bearbeta tillsammans på olika håll i landet, som sedan alla lärare kan ha glädje av, där faktarutor, siffror och listor i innehållet automatiskt uppdateras med data ifrån myndigheter för att hållas aktuella.
Tänk om räkneexempel använde data som handlar historia och data som har en relevans för ens framtid och nutid.
Tänk om det var enkelt att skriva ut faktablad kring smärtlindring till gravida med faktaruta med antal kvinnor som får besvär efteråt, och hur många som använt vilken smärtlindring var färska uppgifter.

Lärare, som är inloggade och vars identitet är bekräftad, kan enkel gå in och ändra texten för att förbättra den för alla, alternativt justera den till ens egna lektionsplanering lokalt.

Denna sida använder kanske data ifrån SLU, och när/om vi gör det så gäller detta: "”Genom att använda denna applikation åtar jag mig att följa regler för användning av ArtDatabankens information, inklusive att respektera tredje mans upphovsrätt. Detta åtagande görs även direkt mot SLU. Vid brott mot detta kan jag förlora rätten att använda applikationen”."
